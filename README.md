# quiz-bot
Инструмент для оценки продуктовой зрелости команды и выдачи методологии под данную команду.

Данный продукт был разработан на основе заранее составленного тестирования. 
Также был составлен список актуальных методологий работы в команде.
Далее была составлена матрица соразмерности набранных баллов и рекомендуемой методологии.
Таким образом, разработанный бот позволяет пройти тестирование и получить рекомендацию по выбору методологии работы в команде.
